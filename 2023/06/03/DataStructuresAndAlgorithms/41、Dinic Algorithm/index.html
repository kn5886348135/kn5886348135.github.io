  dinic 算法

dinic算法

  1. 无向图是特殊的有向图
  2. 最大网络流算法是解决有向图问题的，也可以解决无向图问题
  3. 深度优先遍历可能会得不到正确的答案
  4. dinic算法补一个反向边，不会让答案变大，但是会把错过的正确答案找回来
  5. 用BFS建立高度，然后DFS，避免在同一层来回走
  6. 用一个数组记录已经使用了的线路
